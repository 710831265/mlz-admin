# Icon 图标

**通过指定 iconfont 的 html class 类名来显示对应的`📦 图标`**

## 内置 icon 列表

```tsx
/**
 * inline: true
 */
import React from 'react';
import IconClasses from '../_site/icon-classes';

export default () => <IconClasses />;
```

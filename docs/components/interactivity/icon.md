# Icon 图标

**通过指定 iconfont 的 html class 类名来显示对应的`📦 图标`**

## 1.内置 icon 列表

```tsx
/**
 * inline: true
 */
import React from 'react';
import IconClasses from './../../demo/icon/icon-classes';

export default () => <IconClasses />;
```
